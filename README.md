## Build

Run:

```
apt install git python3-pip

# with VENV
apt install python3-virtualenv 
bash venv.sh

# WITHOUT
pip install -r requirements.txt

# USAGE: bash html.sh <BUILD-DIR> <DOMAIN> 
bash html.sh ./build/ gyb.docs.oxl.app
```
